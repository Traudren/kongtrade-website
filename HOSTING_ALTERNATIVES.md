# Бесплатные альтернативы Netlify

## Сравнение платформ для хостинга

### 1. **Vercel** ⭐ (Лучший выбор для Next.js)
- **Бесплатный план:** 
  - Неограниченные деплои
  - 100 ГБ bandwidth/мес
  - Автоматические SSL
  - Preview deployments
  - Serverless Functions (100GB-hours/мес)
- **Особенности:** 
  - Оптимизирован специально для Next.js
  - Очень быстрый CDN
  - Edge Functions
  - Отличная интеграция с Git
- **Ограничения:** 
  - Serverless функции имеют таймауты (10 сек на бесплатном плане)
- **Сайт:** https://vercel.com

---

### 2. **Cloudflare Pages** ⭐ (Хороший выбор для статики + CDN)
- **Бесплатный план:**
  - Неограниченные сайты
  - 500 сборок/мес
  - Неограниченный bandwidth
  - Автоматические SSL
  - Cloudflare Workers (100k запросов/день бесплатно)
- **Особенности:**
  - Один из самых быстрых CDN в мире
  - DDoS защита встроена
  - Workers для serverless функций
  - Интеграция с Git
- **Ограничения:**
  - 500 сборок/мес (обычно достаточно для небольших проектов)
- **Сайт:** https://pages.cloudflare.com

---

### 3. **GitHub Pages**
- **Бесплатный план:**
  - Полностью бесплатно для публичных репозиториев
  - Неограниченные деплои
  - 1 ГБ хранилища
  - 100 ГБ bandwidth/мес
- **Особенности:**
  - Простая интеграция с GitHub
  - Автоматический деплой через GitHub Actions
- **Ограничения:**
  - Только статические сайты (нет API routes)
  - Нет serverless функций
- **Сайт:** https://pages.github.com

---

### 4. **Render**
- **Бесплатный план:**
  - Статические сайты: полностью бесплатно
  - Web services: $5 кредитов/мес
  - Автоматические SSL
  - Неограниченный bandwidth для статики
- **Особенности:**
  - Простой деплой
  - Поддержка Docker
  - Автоматические SSL
- **Ограничения:**
  - Web services "засыпают" после 15 мин неактивности на бесплатном плане
- **Сайт:** https://render.com

---

### 5. **Railway**
- **Бесплатный план:**
  - $5 кредитов/мес
  - Автоматические SSL
- **Особенности:**
  - Хорош для full-stack приложений
  - Поддержка баз данных
  - Простой деплой
- **Ограничения:**
  - Ограниченные кредиты (быстро расходуются)
- **Сайт:** https://railway.app

---

## Рекомендации для вашего проекта (Next.js)

### Если нужны API routes и serverless функции:
1. **Vercel** - лучший выбор, оптимизирован для Next.js
2. **Cloudflare Pages + Workers** - отличная альтернатива с мощным CDN

### Если только статический сайт:
1. **GitHub Pages** - самый простой вариант
2. **Cloudflare Pages** - для лучшей производительности
3. **Vercel** - тоже отлично работает

### Для production с высоким трафиком:
1. **Cloudflare Pages** - неограниченный bandwidth
2. **Vercel** - 100 ГБ обычно достаточно для начала

---

## Миграция с Netlify

### На Vercel:
```bash
# Установить Vercel CLI
npm i -g vercel

# Деплой
vercel
```

### На Cloudflare Pages:
1. Подключить GitHub репозиторий в Cloudflare Dashboard
2. Настроить build settings:
   - Build command: `npm run build`
   - Build output directory: `.next`
   - Node version: установить в package.json

---

## Вывод

**Для Next.js проекта лучше всего подойдет Vercel** - он создан командой Next.js и максимально оптимизирован.

**Если нужен максимальный bandwidth** - Cloudflare Pages с неограниченным трафиком.

**Если проект статический** - GitHub Pages самый простой вариант.
