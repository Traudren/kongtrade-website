# Настройка DNS для kongtrade.com на Vercel

## Шаг 1: Добавление домена в Vercel

1. Войдите в [Vercel Dashboard](https://vercel.com/dashboard)
2. Выберите ваш проект (kongtrade-website)
3. Перейдите в **Settings** → **Domains**
4. Нажмите **Add Domain**
5. Введите `kongtrade.com` и нажмите **Add**
6. Также добавьте `www.kongtrade.com` (опционально, но рекомендуется)

## Шаг 2: Получение DNS записей от Vercel

После добавления домена Vercel покажет вам **точные DNS записи**, которые нужно добавить в Namecheap.

Обычно это:
- **A запись** для `@` (корневой домен) → IP адрес Vercel
- **CNAME запись** для `www` → `cname.vercel-dns.com` (или похожий)

**ВАЖНО:** Vercel покажет вам уникальные значения для вашего проекта. Используйте именно их!

## Шаг 3: Настройка DNS в Namecheap

### Вариант A: Использование DNS серверов Vercel (рекомендуется)

1. Войдите в [Namecheap](https://www.namecheap.com)
2. Перейдите в **Domain List** → выберите `kongtrade.com`
3. Нажмите **Manage** рядом с доменом
4. В разделе **Nameservers** выберите **Custom DNS**
5. Добавьте DNS серверы от Vercel (обычно это что-то вроде):
   - `ns1.vercel-dns.com`
   - `ns2.vercel-dns.com`
   
   **НО:** Vercel покажет вам точные значения в панели Domains!

### Вариант B: Использование DNS записей (если оставляете Namecheap DNS)

1. В Namecheap → **Domain List** → `kongtrade.com` → **Manage**
2. Перейдите в **Advanced DNS**
3. Удалите все существующие A и CNAME записи для `@` и `www`
4. Добавьте новые записи согласно инструкциям Vercel:

   **Для корневого домена (kongtrade.com):**
   - Type: **A Record**
   - Host: `@`
   - Value: IP адрес от Vercel (например, `76.76.21.21`)
   - TTL: Automatic (или 3600)

   **Для www (www.kongtrade.com):**
   - Type: **CNAME Record**
   - Host: `www`
   - Value: `cname.vercel-dns.com` (или значение от Vercel)
   - TTL: Automatic (или 3600)

## Шаг 4: Проверка настроек

### В Vercel:
1. В панели **Domains** проверьте статус домена
2. Должен быть статус **Valid Configuration** или **Pending** (если DNS еще распространяются)

### Проверка DNS записей:
Используйте онлайн инструменты для проверки:
- [whatsmydns.net](https://www.whatsmydns.net) - введите `kongtrade.com`
- [dnschecker.org](https://dnschecker.org) - проверьте A и CNAME записи

### Через терминал:
```bash
# Проверка A записи
dig kongtrade.com +short

# Проверка CNAME для www
dig www.kongtrade.com +short
```

## Шаг 5: Ожидание распространения DNS

- **Обычно:** 5-30 минут
- **Максимум:** до 48 часов (но обычно быстрее)
- После распространения Vercel автоматически выдаст SSL сертификат

## Частые проблемы и решения

### Проблема: "Invalid Configuration" в Vercel

**Решение:**
1. Убедитесь, что используете **точные значения** от Vercel (не примеры из интернета)
2. Проверьте, что в Namecheap нет конфликтующих записей
3. Удалите старые A/CNAME записи перед добавлением новых

### Проблема: DNS записи не распространяются

**Решение:**
1. Подождите до 48 часов
2. Очистите DNS кэш: `ipconfig /flushdns` (Windows) или `sudo dscacheutil -flushcache` (Mac)
3. Попробуйте другой DNS сервер (например, Google DNS: 8.8.8.8)

### Проблема: Сайт работает, но без SSL

**Решение:**
1. Подождите - Vercel автоматически выдаст SSL сертификат через несколько минут после успешной настройки DNS
2. Проверьте в Vercel → Domains → SSL статус

## Важные моменты

1. **Не используйте примеры DNS записей** - всегда используйте значения, которые показывает Vercel для вашего проекта
2. **Один домен = один набор DNS записей** - не смешивайте разные методы
3. **www и корневой домен** - настройте оба для полной поддержки
4. **Проверка в Vercel** - после настройки DNS проверьте статус в панели Vercel
